# 数据流与契约设计

> 跨模块数据传递、类型契约、解码边界的通用原则。

## 已沉淀

- **叶子字段契约在解码边界归一化** — 生成式系统的 schema 防线分层：入口 prompt 降错误率，解码边界修可修复形状，最终模型只拦截漏网数据
- **抽取层级对象：标题角色 > 标题层级** — 集合名负责切 range，实例名负责落字段；不能把父级责任组标题误抽为具体实例
- **队列消息区分 envelope 和 payload** — 调度元数据在 Worker 边界剥离，只把业务 payload 传给领域函数
- **滑动窗口按时间定义成员资格** — 惰性剪枝比维护窗口大小更优雅；时间决定成员，容量只是防爆保护
